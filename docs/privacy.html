개인정보 처리방침

최종 개정: 2026-07-20

본 방침은 YT Separator 웹사이트 (ytseparator.com) 및 데스크톱 앱과 관련하여 어떤 정보가 어떻게 처리되는지 설명합니다. 앱의 오디오·영상 데이터는 서버로 전송되지 않고 사용자 컴퓨터에서 처리됩니다.

1. 수집하는 정보

웹사이트 (ytseparator.com)

  • 광고 관련: Google AdSense 가 표준 광고 쿠키·식별자를 통해 방문·상호작용 데이터를 수집합니다. 우리 서버는 이 데이터를 저장하지 않습니다.
  • 언어 선택: 브라우저 localStorage 에 언어 설정 (yss:site-lang) 을 저장합니다. 서버 전송 X.
  • 서버 로그: 사이트는 GitHub Pages 로 호스팅되며 Cloudflare 프록시를 거칩니다. IP · User-Agent 등 표준 웹 서버 로그가 일시적으로 기록될 수 있습니다.

데스크톱 앱 (YT Separator)

  • 수집 없음: 앱은 텔레메트리·분석·사용 기록을 원격 서버로 전송하지 않습니다.
  • 로컬 저장: 라이브러리·설정·처리 결과는 사용자 컴퓨터에만 저장됩니다.
  • 업데이트 확인: 새 버전 확인을 위해 GitHub Releases API 에 접속합니다 (IP · User-Agent 만).
  • YouTube 다운로드: 사용자가 요청한 URL 에서만 콘텐츠를 받아옵니다 (yt-dlp 사용). 어떤 URL을 처리했는지는 기록되지 않습니다.

2. 정보의 사용

  • AdSense 데이터: Google 이 광고 표시·측정·개인화에 사용합니다. 자세한 내용은 Google 개인정보처리방침 참조.
  • 언어 설정: 다음 방문 시 언어를 기억하기 위해 사용됩니다.
  • 서버 로그: 어뷰징·오류 조사 목적. 개별 방문자 식별을 위해 분석되지 않습니다.

3. 제3자 서비스

  • Google AdSense — 광고 표시. Google 개인정보처리방침
  • GitHub — 사이트 호스팅 · 릴리즈 배포. GitHub Privacy
  • Cloudflare — 도메인 프록시 · CDN. Cloudflare Privacy

4. 쿠키 · 저장 데이터

  • AdSense 쿠키: 광고 개인화·측정 목적. Google 광고 정책
  • localStorage: 웹사이트 · 앱의 사용자 설정 (언어 · UI 상태). 브라우저 저장소에만 존재.
  • 웹사이트는 자체 tracking 쿠키를 두지 않습니다.

5. 사용자 권리

  • 광고 개인화 해제: Google Ads Settings 에서 언제든 조정 가능.
  • localStorage 삭제: 브라우저 설정에서 사이트 데이터 삭제 시 초기화.
  • 앱 데이터 삭제: 앱 삭제 시 · 사용자 데이터 폴더 (%APPDATA%\yt-separator-desktop) 를 수동 삭제.

6. 아동 정보 보호

본 서비스는 만 13세 미만 아동으로부터 개인정보를 의도적으로 수집하지 않습니다. 만약 만 13세 미만 사용자가 정보를 제공했다는 사실을 확인하는 경우 즉시 해당 정보를 삭제합니다.

7. 방침 개정

본 방침은 서비스 변경·법적 요구에 따라 개정될 수 있습니다. 개정 시 이 페이지 최상단의 "최종 개정" 일자가 갱신됩니다. 중요한 변경 시 앱·사이트에 별도 공지할 수 있습니다.
