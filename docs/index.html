로컬 처리

4-stem · 6-stem

htdemucs · htdemucs_6s 두 모델. 6-stem 은 기타 · 피아노를 추가 분리합니다.

WebGPU 가속

가능하면 GPU 로 처리해 CPU 대비 훨씬 빠릅니다. 실패 시 CPU 자동 폴백.

영상 sync 재생

영상과 스템 오디오가 정확히 sync 되어 재생. seek · pause · 배속 모두 대응.

연습 도구

키 변경 ±6 반음, 배속(피치 보존), A-B 구간 반복, Solo 로 원하는 악기만 감상.

라이브러리

처리한 영상을 저장 · 재사용. 즐겨찾기, 그룹, 4↔6-stem 재분리, 스템 저장 지원.

자주 묻는 질문

모델을 한 번 다운로드하면 이후 로컬 파일은 완전 오프라인으로 처리됩니다. YouTube 링크 다운로드는 인터넷이 필요합니다.

현재 Windows 만 지원합니다. macOS · Linux 는 향후 검토 예정.

GPU (WebGPU) 지원 시스템에서는 3분 곡을 30초~2분 정도에 처리합니다. CPU 는 시스템에 따라 3~10분.

개인 · 비상업적 사용은 무료입니다. 상업적 재배포 · 통합은 별도 문의.

코드 서명 인증 절차가 진행 중이라 SmartScreen 이 경고를 표시할 수 있습니다. 자체 서명된 인증서는 사용 중이며, "추가 정보" → "실행" 으로 진행하시면 됩니다.
