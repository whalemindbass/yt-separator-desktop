기능

로컬 처리

영상·오디오가 서버로 전송되지 않고 사용자 컴퓨터에서 온전히 처리됩니다.

4-stem · 6-stem

htdemucs / htdemucs_6s 두 모델. 6-stem은 기타·피아노를 추가로 분리합니다.

WebGPU 가속

가능한 시스템에서는 GPU로 처리해 CPU 대비 훨씬 빠릅니다. 실패 시 CPU 자동 fallback.

영상 sync 재생

영상과 스템 오디오가 정확히 sync되어 재생. seek·pause·재생속도 변경 모두 대응.

키 변경

반음 단위 ±6까지 피치 이동. 드럼은 원본 유지해 리듬 자연스러움.

라이브러리

처리한 영상을 저장·재사용. 즐겨찾기, 그룹, 이름 편집, 4↔6-stem 재분리 지원.
